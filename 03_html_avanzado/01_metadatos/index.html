metadatos

  • html lang="es" lang=es. Con este metadato, asigna el idioma del texto de la página.
  • meta charset="UTF-8" Con este metadato, asigna los caracteres que se van a escribir en la página.
  • meta name="viewport" content="width=device-width, initial-scale=1.0"> Con este metadato definimos las propiedades de visualización de la página
  • width=device-width: Esto significa que el ancho de la página será igual al ancho del dispositivo
  • initial-scale=1.0 Cuando vayamos a una página, se reestablece la vista de la página al tamaño original, si le hemos hecho zoom, cuando volvamos a ir a ella, aparecerá con un tamaño normal.
  • title> metadatos /title>dentro de la etiqueta tittle escribimos el nombre que aparece en la parte superior de la página, junto al metadato.
  • meta name="description" content=""> Con este metadato dentro de las comillas del content, se escribe una descripción de la página con menos de 100 caracteres
  • meta name="keywords" content=""> Con este metadato dentro de las comillas del content, se escriben las palabras clave separadas por comas que servirán al usuario para encontrar en google la página
  • meta name="autor" content="El nombre del autor"> Con este metadato dentro de las comillas del content, se escribe el nombre del autor
  • meta name="robots" content="nofollow">Con este metadato dentro de las comillas del content, se le indica a los navegadores que no muestren fragmentos de nuestra página.
  • meta name="robots" content="nosnippet">Con este metadato dentro de las comillas del content, se le indica a los navegadores que no indexen o muestren la página en los resultados de busqueda
  • meta name="robots" content="noarchive">Con este metadato dentro de las comillas del content, para que no se guarde una copia de la página el el caché del navegador.
  • meta name="robots" content="noimageindex">Con este metadato dentro de las comillas del content, no se indexan las imágenes, no aparecen en el navegador.
  • meta name="OG:title" content=""> Con este metadato dentro de las comillas del content, hay muchas variedades de significados, entre ellos sirven para que cuando se busque una página en internet, aparezca el nombre de la página, una foto y una pequeña descripción.
