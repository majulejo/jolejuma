TEXTO

Podemos definir un color principal --color-principal:#851; o como lo queramos llamar y luego para aplicarlo podemos llamarlo con la función var() de la siguiente manera. Por ejemplo si queremos modificar un color de texto le podemos llamar color: var(--color-principal);

Este P como está fuera de los box y box2, no le afectan las variables de color, por eso no tiene el color de las cajas, ni le funciona el color del borde.
